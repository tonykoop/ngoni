# Ngoni — Learn to Play

Starter songbook for Tony Koop's ngoni build.

## About the ngoni

The ngoni (also n'goni, ngòni) is a West African lute from the Mande tradition.
It is an ancestor of the banjo. The instrument is fretless, plucked, and uses
a pentatonic scale. The number of strings varies (3–7); this songbook assumes
a **5-string ngoni** tuned to **D pentatonic**.

## Tuning

**5-string D pentatonic (low to high):**

| String | Pitch |
|--------|-------|
| 5 (lowest, thumb) | D3 |
| 4 | F♯3 |
| 3 | A3 |
| 2 | D4 |
| 1 (highest) | F♯4 |

The thumb plays the bass drone strings (4 and 5); the fingers play the melody
strings (1, 2, 3). This separation is the foundation of ngoni technique.

## Notation

Staff notation in treble clef at sounding pitch. No fret positions (fretless).
Open strings are marked with `0`; slides are marked `sl.`.

## Technique

- **Thumb:** plays downward strokes on the bass strings (strings 4, 5).
- **Index + middle fingers:** pluck melody strings upward (strings 1, 2, 3).
- **Slide:** the fretless neck allows gliding between pitches. Slides are
  idiomatic — try a quick upward slide into any melody note for character.
- **Drone:** string 5 (D3) is often left ringing as a drone under the melody.
