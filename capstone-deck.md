# Title
Ngoni L2/L2+ packet.

# Project Intent
Document a respectful engineering prototype for a Mande bowl-lute family using computed string schedules, segmented bowl construction, bridge planning, and open prototype validation.

# Readiness
L2/L2+ documentation packet. CAD model, prototype photos, measured tuning data, `.pptx`, and `.pdf` exports are pending.

# Cultural Boundary
The packet includes provenance language and does not claim cultural authority or traditional certification.

# Variants
Donso 6-string, Kamele 10-string, and Kamele 14-string schedules are documented with the Kamele 10 as the primary build.

# Physics
Mersenne-Taylor string physics is used; no unrelated NAF K2 or beam corrections are applied.

# Resonator
The skin-headed segmented bowl is treated as a coupled resonator and radiator with prototype measurements still required.

# Manufacturing
The packet includes segmented bowl, bridge, neck, skin, CNC, and drawing notes.

# Validation
Validation covers string safety, neck deflection, bridge stability, bowl/head construction, and public-release gates.

# Risks
Main risks are bridge stability, Kamele 14 treble string fragility, bowl joints, skin tension, and cultural wording.

# Shop Evidence
Needed evidence includes CAD, prototype photos, measured tuning, and bridge/head load observations.

# Next Actions
Generate binary deck/print artifacts, build CAD, prototype the Kamele 10, and record measured validation data.
